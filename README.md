## Setup

**INSTALL PACKEAGES**
```docker compose run --rm cartrader yarn install```

**RUN BUILD**
```docker compose build```

## Development Server

**RUN DEV**
``` docker compose up ```

Check out the [deployment documentation](https://nuxt.com/docs/getting-started/deployment) for more information.
